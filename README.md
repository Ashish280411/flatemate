# FlatmateAI

FlatmateAI is a smart accommodation matching web app built for students and professionals who want to find compatible flatmates, not just available rooms.

The platform focuses on lifestyle compatibility by comparing factors such as budget, preferred area, gender preference, sleep schedule, and daily habits. Instead of matching people only by rent and location, FlatmateAI aims to help users make better shared-living decisions with AI-based compatibility insights.

## Project Idea

Many people struggle to find a flatmate who matches their lifestyle. A low-rent room is not enough if routines, habits, and expectations clash. FlatmateAI solves this by creating structured preference profiles and using compatibility scoring to suggest better roommate matches.

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Ashish280411/flatemate.git
cd flatemate
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```

## Available Scripts

- `npm run dev` starts the Vite development server
- `npm run build` creates the production build
- `npm run preview` previews the production build locally
- `npm run lint` runs ESLint
